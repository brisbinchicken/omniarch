# OmniArch ⚡️

An interactive, sysadmin-grade Arch Linux post-installation script and dotfile manager. 

Built with `gum` for a modern terminal UI, OmniArch allows you to cherry-pick your core system packages, terminal utilities, networking tools, and desktop environments on a fresh install.

## 🚀 Installation

Run this on a fresh, barebones Arch Linux installation (do not run as root):

```bash
curl -sL [https://raw.githubusercontent.com/brisbinchicken/omniarch/main/setup.sh](https://raw.githubusercontent.com/brisbinchicken/omniarch/main/setup.sh) | bash
